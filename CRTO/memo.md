# Initial Access
フィッシングペイロードの分類法
DELIVERY(CONTAINER(TRIGGER + PAYLOAD + DECOY))

- Delivery: deiliver the package to the victim
- Container: container format used to package the files
- Trigger: trigger payload execution
- Payload: malicious code to execute
- Decoy: a file to display to the vimctim
